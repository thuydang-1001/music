Pizza Sales Data Analyst  Project

🔹 Project Overview

This is a complete data analyst portfolio project using SQL and Power BI, where we go from basic to advanced and build a step-by-step solution. The project focuses on analyzing pizza sales data and creating a complete interactive dashboard to answer real business questions based on customer requirements.

🔹 Tools Used
SQL (MS SQL Server)
Power BI
Excel (CSV data source)

🔹 Project Workflow
1. SQL Data Analysis
Imported raw data into MS SQL Server and created a database
Fired SQL queries based on problem statement and customer requirement
Calculated KPIs such as:
Total Revenue
Average Order Value
Total Orders
Total Pizza Sold
Used aggregations, distinct count, and grouping to get required results
Saved queries and outputs in a document for future reference
2. Data Validation
Performed two-way validation by comparing SQL query results with Power BI dashboards
Ensured that the numbers shown in dashboards are "pinpoint and very much correct"
3. Power BI Dashboard
Connected Power BI to SQL Server (or CSV file)
Performed data cleaning and transformation using Power Query
Used DAX functionalities, conditional filtering, and formatting
Built a dynamic and interactive dashboard

Dashboard includes:

KPIs (Total Revenue, Total Orders, Average Order Value, Total Pizza Sold)
Daily trend for total orders (bar chart)
Monthly trend for orders (line chart)
Percentage of sales by pizza category and size (donut charts)
Total pizza sold by category (funnel chart)
Top 5 and bottom 5 best/worst sellers

🔹 Key Insights
Identified busiest days and times based on order trends
Analyzed sales performance by category and size
Found top 5 best sellers and bottom 5 worst sellers
Helped understand which products are performing best or underperforming

🔹 Business Impact
Supports business decisions and marketing actions
Helps identify underperforming products and opportunities for improvement
Provides a clear and interactive view of sales performance
